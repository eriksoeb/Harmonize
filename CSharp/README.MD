
## Build Executable (Windows Forms – C#)

The `WindowsFormsAppCSharp19` folder contains a Windows Forms application that can be opened and built using **Microsoft Visual Studio**.

---

### Prerequisites

- Microsoft Visual Studio (2019 or later recommended)
- .NET Framework version compatible with the project (installed via Visual Studio)

---

### Build Steps

1. Open **Visual Studio**
2. Select **File → Open → Project/Solution**
3. Browse to the `WindowsFormsAppCSharp19` folder
4. Open the solution file (`.sln`) or the project file (`.csproj`)

---

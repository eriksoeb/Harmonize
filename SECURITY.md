# Security Overview

## 1. Access Control
Harmonize implements role-based access control and user access control to ensure that users can only access data and functions authorized by the implementing institution.

## 2. Authentication and Authorization
The platform supports configurable authentication mechanisms and enforces authorization rules at application and data levels.

## 3. Audit and Logging
Harmonize supports audit logging to track data access and system actions, supporting accountability and transparency.

## 4. Deployment Security
Harmonize can be deployed on-premise or in controlled cloud environments, allowing institutions to apply their own infrastructure security policies.

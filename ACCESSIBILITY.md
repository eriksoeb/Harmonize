# Accessibility Statement

Harmonize is committed to inclusive and accessible design.

The user interface is developed to align with the Web Content Accessibility Guidelines (WCAG) 2.1, supporting:
- Keyboard navigation
- Screen reader compatibility
- Clear visual hierarchy and contrast

Harmonize is designed to support compliance with accessibility regulations, including:
- Americans with Disabilities Act (ADA) of 1990
- EU Web Accessibility Directive (Directive (EU) 2016/2102)

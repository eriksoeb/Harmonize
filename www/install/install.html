Harmonize is a Windows application for exploring, analyzing and comparing time-series data.
The zip contains all files needed to run and test the application with demo data.
(Harmonize will require your own MS-SQL database with your data for professional use)

Download Harmonize Installer
Download Harmonize SqlAddIn Installer

Installation steps:

  1. Download Harmonize.Setup.zip, run Setup.exe (requires admin rights) and install into the default folder: C:\Harmonize\App.
  2. Start Harmonize.exe or the shortcut on the Desktop
  3. SQL Add-in: Download and run the setup installer (as administrator) into the folder: C:\Harmonize\App.
  4. Workaround: The Harmonize app can be downloaded and copied to C:\Harmonize\App and started from there without admin rights on most computers.
